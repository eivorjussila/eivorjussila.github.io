How do I design my portfolio?

Cloudy day, Nov 15, 2018

Hello, so here is how I make this portfolio. First of all, I read the book How to design programs edition 2. I have been mentioning this book so many times but I honestly believe it deserves to be repeated. Second, I use the free online ebook "select star sql" to learn about sql. Third, I go on w3schools website and start to learn html, css and js. During the process of making the websites, I use public domain images from pexels and color suggestions on colorpalettes website.
